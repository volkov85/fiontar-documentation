  • Installing the theme

    By far, the easiest way to install the theme is through the WordPress dashboard.

    First, you’ll need to download the theme from ThemeForest from your download section.

    Afterward, navigate to "Appearance" - "Themes" - "Add New".

    And after the page is fully loaded you’ll notice right next to the top "Add themes" title, a button that says "Upload theme". Click it and a “browse field” will appear. Browse for the theme’s zip and that’s it.

    You may then activate the theme as you would any other WordPress theme.

    Alternatively, you may choose to install the theme manually. To do so, download the latest version from ThemeForest from your download section, then upload the extracted folder to the "/wp-content/themes/" on your server via FTP. You will now have access to activate the theme from the "Appearance" - "Themes" screen in your dashboard.

  • How to update the theme

    The easiest way to manually update your theme is unpack a new version of our theme inside "\wp-content\themes\" directory.

    Or you can use "Easy Theme and Plugin Upgrades" plugin.

  • How to add Gutenberg blocks

    To add a block you need to press "Add block" and choose one. All custom blocks can be found in the "Fiontar blocks" section.

  • How to change settings in Gutenberg blocks

    To change settings you need to select a block and press "Settings" button. In the "Settings" panel select "Block" tab.

  • How to insert inner blocks

    To insert inner blocks you need to press "Add block" inside the Gutenberg block than support inner blocks.

  • How to delete Gutenberg blocks

    To delete blocks you need to click on the block than "More options" - "Remove Block".

  • How to add a favicon image

    To add a favicon image, in backend simply access "Appearance" - "Customize" and access the Site Identity group. Here you’ll notice the "Browse image" button. WordPress recommends a 512px tall/wide image which will be resized automatically for desktop use and smart devices screens.

  • How to create a menu

    To create a menu you first need to go to the "Appearance" - "Menus" page and create one. The following tutorial will guide you through all the process: https://codex.wordpress.org/WordPress_Menu_User_Guide

    We have 3 kind of menu in our demo content.

  • How to add Fiontar widgets

    To add a widget you first need to go to "Appearance" - "Widgets".

    Select from "Avaliable Widgets" - "Feedback Widget" or "Subcribe Widget" than drag and drop them in the "Footer horizontal widget area".

  • How to add Fiontar testimonials

    To add a testimonial you first need to go to "Testimonials" - "Add new". Than fill all fields and press "Publish".

    Go to "Appearance" - "Widgets". Make sure that you have "Feedback widget" in the "Footer horizontal widget area".

    You can set "Number of Testimonials to Show" and press "Save".

  • How to install a demo content

    To install a demo content you first need to unpack 3 files: fiontar-demo-content.xml, fiontar-widgets.wie and fiontar-customizer.dat.

    Then go to "Appearance" - "Import Demo Data" ("One Click Demo Import" plugin must be installed).

    Select our 3 files in the interface and press "Import Demo Data".

    Finally, go to "Settings" - "Reading", press on the radio button "A static page (select below)" and select "Homepage" - "Main" in the combobox. Don't forget to press "Save changes".
